## 简介

Kronos是一个开源的金融K线基础模型，专为金融市场的"语言"设计。该项目提供了完整的预测、微调和部署工具链，支持多种操作系统和硬件配置。本指南将详细介绍如何在不同环境中安装和配置Kronos项目。

## 系统要求

### Python版本要求
- **最低要求**: Python 3.10+
- **推荐版本**: Python 3.10.x 或 3.11.x
- **不支持**: Python 3.8、3.9（存在兼容性问题）

### PyTorch版本兼容性
- **最低要求**: PyTorch ≥ 2.0.0
- **推荐版本**: PyTorch 2.0.x 或更高版本
- **WebUI特殊要求**: WebUI模块需要PyTorch ≥ 2.1.0

### 操作系统支持
- **Linux**: 完全支持，推荐Ubuntu 18.04+
- **Windows**: 完全支持，推荐Windows 10/11
- **macOS**: 部分支持，主要功能可用，GPU加速受限

## 核心依赖项详解

### 主要依赖包作用

#### numpy
- **版本**: 最新稳定版
- **作用**: 数值计算基础库，金融数据处理的核心
- **性能影响**: 影响所有数值运算速度

#### pandas
- **版本**: 2.2.2（固定版本）
- **作用**: 数据结构和数据分析工具
- **用途**: K线数据加载、时间序列处理、数据预处理

#### torch
- **版本**: ≥2.0.0（≥2.1.0用于WebUI）
- **作用**: 深度学习框架，模型训练和推理
- **特性**: 支持GPU加速、自动微分、分布式训练

#### huggingface_hub
- **版本**: 0.33.1（固定版本）
- **作用**: Hugging Face模型仓库访问
- **功能**: 模型下载、上传、版本管理

#### matplotlib
- **版本**: 3.9.3（固定版本）
- **作用**: 数据可视化
- **用途**: 预测结果图表绘制

#### einops
- **版本**: 0.8.1（固定版本）
- **作用**: 张量操作工具
- **应用**: 注意力机制实现、张量重排

#### tqdm
- **版本**: 4.67.1（固定版本）
- **作用**: 进度条显示
- **用途**: 训练过程监控、数据加载进度

#### safetensors
- **版本**: 0.6.2（固定版本）
- **作用**: 安全的Tensor存储格式
- **优势**: 更快的加载速度、内存安全

### 内存优化技巧

#### 混合精度训练
```python
from torch.cuda.amp import autocast, GradScaler

scaler = GradScaler()

for data, target in dataloader:
    optimizer.zero_grad()
    
    with autocast():
        output = model(data)
        loss = criterion(output, target)
    
    scaler.scale(loss).backward()
    scaler.step(optimizer)
    scaler.update()
```

#### 梯度累积
```python
accumulation_steps = 4
for i, (data, target) in enumerate(dataloader):
    loss = compute_loss(data, target)
    loss = loss / accumulation_steps
    loss.backward()
    
    if (i + 1) % accumulation_steps == 0:
        optimizer.step()
        optimizer.zero_grad()
```
