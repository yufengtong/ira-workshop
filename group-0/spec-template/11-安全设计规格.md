## 3. 数据安全

> 定义数据分类分级、加密策略、存储安全规范。**投研场景核心关注点**。

### 3.1 数据分类与分级

| 数据类型 | 安全等级 | 示例 | 保护措施 |
|----------|----------|------|----------|
| 研报原文 | **机密** | PDF/HTML 研报文件 | 加密存储，访问审计 |
| 问答记录 | **内部** | 用户提问与AI回答 | 存储加密，定期清理 |
| 用户信息 | **内部** | 用户名、角色、操作日志 | 脱敏展示，最小采集 |
| 系统配置 | 一般 | 模型参数、界面配置 | 版本控制 |

### 3.2 加密策略

| 场景 | 加密方式 | 说明 |
|------|----------|------|
| 传输加密 | 【如 TLS 1.2+】 | 所有 API 通信强制 HTTPS |
| 存储加密 | 【如 AES-256】 | 研报文件与问答记录落盘加密 |
| 密钥管理 | 【如 环境变量 / KMS】 | 密钥不硬编码，定期轮转 |

## 4. 接口安全

> 与 `09` API 规格配合，定义 API 层面的安全防护措施。

| 防护措施 | 实现方式 | 说明 |
|----------|----------|------|
| 输入校验 | 服务端白名单校验 | 所有用户输入进行类型、长度、格式校验 |
| SQL 注入防护 | 参数化查询 / ORM | 禁止字符串拼接 SQL |
| XSS 防护 | 输出转义 + CSP | 研报内容渲染时防止脚本注入 |
| CSRF 防护 | 【填写】 | 状态变更接口需 CSRF Token |
| 速率限制 | 【如 100 req/min/user】 | 防止接口滥用 |
| 文件上传校验 | 类型白名单 + 大小限制 | 仅允许 PDF/HTML，限制单文件大小 |

## 5. LLM / AI 安全

> 针对大模型集成的专项安全措施。投研场景数据敏感，需严格管控数据流向。

| 风险点 | 防护措施 | 验收标准 |
|--------|----------|----------|
| 数据外泄至第三方模型 | 本地部署模型 / 内网百炼服务，禁止调用外网 API | 网络出口审计零外传记录 |
| Prompt 注入攻击 | 系统 Prompt 隔离，用户输入预处理过滤 | 【验收标准】 |
| 模型幻觉 / 错误信息 | 回答标注来源引用，标识置信度 | 【验收标准】 |
| 敏感信息泄露（模型回复） | 输出过滤层，正则匹配敏感信息并脱敏 | 【验收标准】 |

## 6. 审计与日志

> 定义安全审计日志的记录范围、保留策略。

| 事件类型 | 记录内容 | 保留期限 | 告警 |
|----------|----------|----------|------|
| 登录 / 登出 | 用户ID、时间、IP、结果 | 【如 180天】 | 连续失败 ≥5 次告警 |
| 研报访问 | 用户ID、文档ID、操作类型 | 【如 365天】 | 异常批量下载告警 |
| 数据导出 | 导出内容摘要、目标 | 【如 365天】 | 每次导出需审批 |
| 系统配置变更 | 变更人、变更前后值 | 【如 永久】 | 关键配置变更即时通知 |
