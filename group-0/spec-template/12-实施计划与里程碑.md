## 1. 里程碑（S1～S4）

> **切分原则**：每刀切出一个**可演示的用户价值**。不是"接口写好了"，而是"能给业务看什么"。

**示例 — S1**：

**S1 会话管理**：创建/列表/删除会话 API + 前端会话侧栏 → 满足 REQ-001 部分 → 验证 TC-M01-020～025

| **S1** 会话管理 | 会话 CRUD API + 前端侧栏 | REQ-001 部分 | TC-M01-020～025 |
