## 1. 与总表对应

总表 V2 **05b**；对齐 **`05`、`09`、`07`**。

## 2. 测试层级

| 层级 | 覆盖 | P0 |
|------|------|-----|
| **单元** | 模板变量渲染、规则 `condition` 求值（简版）、`dryRun` 分支 | |
| **集成** | 规则 CRUD；`dispatch` dry-run **不调外网** 或调 **沙箱 webhook**；`deliveries` 落库 | 每条主路径 ≥1 |
| **契约** | `NotifyRule`、`Delivery` 与 `09` 一致 | |
| **E2E** | 创建规则 → 试发（dry）→ 历史有记录 → channel test | US-M4-001～003 |

### 2.1 用例 ID 示例

| TC-ID | 断言 |
|-------|------|
| M4-U-TPL-01 | 缺变量 → 400，不创建 `notify_deliveries` |
| M4-I-RULE-01 | `POST` + `GET` + `PATCH` + `DELETE` 规则闭环 |
| M4-I-DISP-01 | 默认 `dryRun=true` 时外网 HTTP mock **0 次**（或仅沙箱） |
| M4-I-DISP-02 | `dryRun=false` + 合规 stub 通过 → `status=sent` 或渠道返回模拟成功 |
| M4-I-BLOCK-01 | 合规 stub 拦截 → `blocked` + `M4_COMPLIANCE_BLOCK` |
| M4-I-CH-01 | `POST .../channels/{type}/test` 返回 `ok` 或明确错误码 |
| M4-E2E-001 | 运维页完整试发 | US-M4-002 |

## 3. 质量门禁

| 门禁 | 说明 | 阻塞 |
|------|------|------|
| G-M4-LINT | 常规 | 是 |
| G-M4-INT | M4-I-RULE-01、M4-I-DISP-01、M4-I-CH-01 绿 | 是 |
| G-M4-DRYRUN | **prod 流水线禁止** `dryRun=false` 无沙箱（除非显式 env） | 是 |
