# 文字图片形状排列

一个简单实用的网页工具，可以将文字按照图片的形状进行排列，生成独特的文字艺术效果。

## 功能特点

- 🖼️ **图片上传** - 支持点击上传或拖拽上传图片
- ✏️ **自定义文字** - 输入任意文字内容用于排列
- 🎨 **参数调节** - 可调节字体大小、文字间距、阈值等参数
- 💾 **图片下载** - 支持将生成的效果保存为 PNG 图片
- 📱 **响应式设计** - 适配桌面和移动设备

## 使用方法

1. **上传图片**
   - 点击上传区域选择图片，或直接将图片拖拽到上传区域
   - 支持 JPG、PNG、GIF 等常见图片格式

2. **输入文字**
   - 在文本框中输入想要排列的文字内容
   - 建议输入较长文字以获得更好的填充效果

3. **调整参数**
   - **字体大小**：控制文字的大小（8-24px）
   - **文字间距**：控制文字之间的间距（0-10px）
   - **阈值**：控制图片哪些区域会显示文字（0-255），数值越小显示的文字越少

4. **生成效果**
   - 点击"生成效果"按钮，等待处理完成
   - 预览区域会显示文字按图片形状排列的效果

5. **下载图片**
   - 点击"下载图片"按钮，将生成的效果保存为 PNG 文件

## 技术实现

- 纯 HTML/CSS/JavaScript 实现，无需后端服务
- 使用 Canvas API 进行图像处理和文字渲染
- 通过像素扫描算法识别图片形状区域
- 在图片的深色区域填充文字，实现形状排列效果

## 本地运行

由于浏览器安全限制，需要通过本地服务器访问：

```bash
# 使用 Python 启动本地服务器
python3 -m http.server 8080

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8080
```

或者直接双击 `index.html` 文件在浏览器中打开（部分功能可能受限）。

## 注意事项

1. 建议使用轮廓清晰、对比度高的图片效果更佳
2. 较大的图片会自动缩放处理，以保证生成速度
3. 文字越长，填充效果越丰富
4. 阈值参数需要根据图片特点调整，深色图片建议降低阈值
